Default
```js
<Alert>This is an alert!</Alert>
```

Example:
```js
<Alert>This is default alert!</Alert>
<Alert color="#fff" type="primary">This is primary alert!</Alert>
<Alert type="danger">This is danger alert!</Alert>
<Alert type="light">This is light alert!</Alert>
<Alert color="#fff" type="dark">This is dark alert!</Alert>
```
